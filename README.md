## 🔒 Project 1: Password Strength Checker
An interactive security tool designed to evaluate password complexity and calculate brute-force vulnerability risks.

### Core Features:
- **Immediate Risk Evaluation:** Dropped passwords under 8 characters instantly to mitigate brute-force attacks.
- **Pythonic Efficiency:** Implemented character-type verification utilizing optimized short-circuit logic for linear time complexity $O(n)$.
- **Diagnostic Feedback:** Provides explicit, actionable improvement instructions for weak passwords.

## 🔑 Project 2: Cryptographic Engine (Caesar Cipher)
A mathematical transformation tool focused on data confidentiality in transit using Caesar Cipher algorithms.

### Core Features:
- **Integer Shift Logic:** Converts characters to ASCII integers before applying shift math.
- **Precision Wrapping:** Uses modular arithmetic (`% 26`) to seamlessly wrap letters past Z.
- **Case & Special Character Preservation:** Keeps original capitalization intact while preserving spaces and punctuation.
